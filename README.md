# Number Classifier

### Description
This is a simple Python tool that analyzes any number you type in. It tells you two things: if the number is positive, negative, or zero, and if it is even or odd.

 ### What it does:
It looks at the number to see if it is positive, negative, or zero.

 It uses the modulo operator (%) to see if the number is even or odd.

 It prints a final sentence combining both facts about your number.
### How to use it:
Run the script.

Type in any number (like 10, -5, or 0).

Read the result! (Example: "the number -5 is negative and odd")
